A social network that implements adding and deleting friends, authorization, sending messages etc. 

The project is under development. Some features may be unavailable.

Technologies:

- React (class components)
- TypeScript
- Redux
- React-Redux
- React-Router-Dom
- Redux-Thunk
- React-Hook-Form
- Axios
- SASS
